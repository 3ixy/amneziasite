Не могу подключиться к AmneziaFree

Рассмотрим разные случаи:

  1. Не получается подключиться к Amnezia Free сразу после добавления ключа. Приложение выдает ошибку 1100 или 1103.
  2. Удается подключиться к сервису Amnezia Free, но он не работает
  3. Amnezia Free работал раньше, но по какой-то причине перестал работать.

Если при попытке подключения появляется ошибка 1100 или 1103

Значит Amnezia не может получить основную конфигурацию для подключения от наших сервисов.

  • Попробуйте переподключиться к другой сети. Например, если вы пытаетесь подключиться через WIFI, попробуйте отключить его и попробовать подсоединиться через мобильную сеть, или через другую сим карту.
  • Если вы давно получали конфигурацию через Telegram bot, попробуйте запросить новую конфигурацию через приложение.

Если удается подключиться к сервису Amnezia Free, приложение не показывает ошибок, но VPN не работает

Может помочь сброс конфигурации API, в особенности если вы давно получили свою конфигурация. Сбросить API конфигурацию можно в настройках.

После сброса API попробуйте заново подключиться, если это не помогло, попробуйте сбросить API конфигурацию повторно через некоторое время, возможно в данный момент слишком большая нагрузка на серверы.

  • Попробуйте так же перезагрузить устройство
  • Попробуйте поменять тип подключения, если вы подключаетесь через WIFI, попробуйте мобильный интернет и наоборот.

Если не помог сброс API конфигурации при использовании сотовой сети на Android

  • Попробуйте нажать "Подключиться"
  • Подождать, пока Amnezia подключится
  • Включите режим полета на 10 секунд
  • Отключите полетный режим.
