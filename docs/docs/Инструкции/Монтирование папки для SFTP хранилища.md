# Монтирование папки для SFTP хранилища

Инструкция [Как создать файловое хранилище SFTP](sftp.html)

Откройте главный экран Amnezia. Внизу кликните на значок настроек.

![](../../assets/images/sftpmd_ru_1-f6a91c222ab3f22eb035bbb1df5f6ed7.png)

Далее перейдите в раздел “Серверы”.

![](../../assets/images/sftpmd_ru_2-63f59a1ad8a54359d9020abb9bb3b16f.png)

Выберете сервер с файловым хранилищем “SFTP”.

![](../../assets/images/sftpmd_ru_3-afbcc49107c1fbe593bf952f46f4c7d5.png)

Во вкладке "Сервисы" выберете “SFTP”.

![](../../assets/images/sftpmd_ru_4-bc3133c890dde8c1c080b76996064780.png)

Вы попадете в настройки файлового хранилища.

![](../../assets/images/sftpmd_ru_5-f97203227b75d193f14ea36ae94c9c78.png)

Перед тем, как смонтировать диск, установите
[WinFsp](https://github.com/winfsp/winfsp/releases/tag/v2.0) и [SSHFS-
Win](https://github.com/winfsp/sshfs-win/releases).

[WinFsp](https://github.com/winfsp/winfsp/releases/tag/v2.0)

Перейдите по ссылке на гитхаб, пролистайте страницу до конца, выберете файл в
формате **.msi** , как указано на скриншоте.

![](../../assets/images/sftpmd_ru_6-dfa06beda803ba3b7c70ee2a9046914c.png)

[SSHFS-Win](https://github.com/winfsp/sshfs-win/releases)

Перейдите по ссылке на гитхаб, пролистайте страницу до раздела "Assets",
выберете файл в формате .msi, как указано на скриншоте.

![](../../assets/images/sftpmd_ru_7-4129ebee6656328bdeaf2b9ba4747721.png)

После установки нажмите “Смонтировать папку на устройство”.

![](../../assets/images/sftpmd_ru_8-a3cf5d3c0092fa34f12214727fa0d305.png)

Amnezia смонтирует Диск на вашем устройстве, так вы получите доступ к
хранилищу без сторонних приложений.

![](../../assets/images/sftpmd_ru_9-21c21910b5647af7a86bd517f6f18a1c.png)
