# Смена DNS сервера

### Для чего может потребоваться смена DNS?

В некоторых случаях, провайдеры интернет-услуг или правительства могут
блокировать доступ к определенным веб-сайтам или сервисам через свой DNS.
Изменение DNS-сервера может помочь обойти такие блокировки и получить доступ к
ограниченному контенту.

Кроме того, некоторые DNS-серверы могут собирать данные о вашей активности в
Интернете, например, посещенные вами веб-сайты и запросы DNS. Сокрытие своего
DNS-сервера, например, с помощью AmneziaDNS, помогает защитить вашу личную
информацию.

Откройте главный экран. На главном экране внизу кликните на значок настроек.

![](../../assets/images/cd_ru_1-b59589fa0652655e3295259e932d3215.png)

Далее перейдите в раздел “Серверы”.

![](../../assets/images/cd_ru_2-d4fd7a1b4a7f474c6ac74cc0ebac9d33.png)

Выберите сервер, в котором вы хотите поменять DNS.

![](../../assets/images/cd_ru_3-cbc7a6d8c8dc836fd2f1f43b9894f83d.png)

Выберите вкладку "Сервисы", далее Amnezia DNS.

![](../../assets/images/cd_ru_4-e02158cb9412cdb08d6de17416f37002.png)

Кликните "Установить".

![](../../assets/images/cd_ru_5-792e19092db24088e8b002cd5c56fd5d.png)

В дальнейшем, вы сможете менять настройки своего DNS сервера. Для этого
откройте главный экран и нажмите на иконку настроек на нижней панели
приложения.

![](../../assets/images/cd_ru_6-b59589fa0652655e3295259e932d3215.png)

Далее выберите раздел “Соединение”

![](../../assets/images/cd_ru_7-9cd7baabdce64f626594147d98ab3654.png)

Здесь вы сможете выбрать опцию “Использовать Amnezia DNS”, или использовать
другие DNS серверы.

![](../../assets/images/cd_ru_8-d1e8d45408cb3f95943196ec650ff1ce.png)
