# Сбросить конфигурацию API

Для соединения с VPN Amnezia использует две конфигурации первичную и основную.
Первичную конфигурацию выдает telegram-бот или приложение, а основную
конфигурацию приложение получает во время подключения.

Кликните на название сервера на главном экране приложения

![](../../assets/images/rapic_ru_1-82bcc00a2a6d1dcd8e7cb225d42f85a3.png)

Нажмите на шестеренку напротив сервера, в котором вы хотите сбросить API
конфигурацию

![](../../assets/images/rapic_ru_2-4819437385387735c630cc7ea8f10472.png)

Найдите вкладку “Управление” Далее выберите "Сбросить конфигурацию API"

![](../../assets/images/rapic_ru_3-68e340df54523bef30bab5fc5cc9a39f.png)
