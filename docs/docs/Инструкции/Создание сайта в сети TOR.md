# Создание сайта в сети TOR

С помощью Amnezia можно не только создавать VPN на сервере, но и разворачивать
сайты в сети TOR на WordPress.

На главном экране Amenzia кликните на значок настроек.

![](../../assets/images/cst_ru_1-f6a91c222ab3f22eb035bbb1df5f6ed7.png)

Далее перейдите в раздел “Серверы”.

![](../../assets/images/cst_ru_2-d5735e2ac9c9bba8a9e764185385afee.png)

Выберете сервер, на котором вы хотите установить сайт в сети TOR.

![](../../assets/images/cst_ru_3-afbcc49107c1fbe593bf952f46f4c7d5.png)

Во вкладке "Сервисы" выберете “Сайт в сети Tor”.

![](../../assets/images/cst_ru_4-df682146f549287b2b4474495c5df410.png)

Нажмите "Установить".

![](../../assets/images/cst_ru_5-13d86064cd040fad6e9c3379d577b8c8.png)

## Настойка сайта в сети TOR

После успешной установки вы увидите адрес сайта, его нужно скопировать и
вставить в строку браузера [Tor](https://www.torproject.org/).

![](../../assets/images/cst_ru_6-78c2c3679687d6844aa8588dc5d2552a.png)

Выберете язык, нажмите “Продолжить”.

![](../../assets/images/cst_ru_7-1e5f37ed145f4d8484086bf9c443dad3.png)

Придумайте название Сайта, введите имя пользователя, а так же Ваш e-mail, не
забудьте сохранить пароль для входа. Нажмите “Установить WordPress”

![](../../assets/images/cst_ru_8-b585d0b3830bc1a69f3a38eddea898eb.png)

Вы установили WordPress для создания сайта в сети Tor. Далее нажмите “Войти”.

![](../../assets/images/cst_ru_9-8caeb565212c8c488f0bf727342a3eb1.png)

Введите имя пользователя, указанное при регистрации, и пароль.

![](../../assets/images/cst_ru_10-7b4ca9d5e16549410a5cbb27cfdedecd.png)

Вы зашли в консоль для создания сайтов с помощью конструктора WordPress, более
подробную информацию по созданию сайтов на WordPress вы найдете на
[официальном сайте WordPress](https://codex.wordpress.org/) .

![](../../assets/images/cst_ru_11-59c0c22f2b30e24b211705a7232938b1.png)
