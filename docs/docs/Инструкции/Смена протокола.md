# Смена протокола

Amnezia - мультипротокольный клиент и поддерживает почти все современные
протоколы: OpenVPN, WireGuard, ShadowSocks IKv2 и OpenVPN с плагином Cloak,
более подробно о протоколах вы можете прочитать в [описании
протоколов](../protocols-info.html).

Откройте приложение Amnezia. На главном экране внизу кликните на название
вашего сервера (на скриншоте он обозначен как Server2).

![](../../assets/images/cp_ru_1-9cdc8beac4ed1c83cb93d75caba2440e.png)

Нажмите на белый прямоугольник с названием VPN-протокола, как на скриншоте.

![](../../assets/images/cp_ru_2-b9108597b824719e5344138b29e011df.png)

Выберите протокол из списка.

Если протокола, к которому вы хотите подключиться нет в списке, значит с вами
поделились конфигурацией только для подключения, запросите конфигурацию с
нужными протоколами у того, кто с вами ей поделился, или [подключитесь к
собственному серверу](https://amnezia.org/starter-guide).

![](../../assets/images/cp_ru_3-6564cd51687abb1bcd2573e94e7c4c70.png)

Вернитесь на главный экран, нажмите кнопку “Подключиться”.

![](../../assets/images/cp_ru_4-ecc4bc33c885d2c94c21ea4defc49650.png)
