# Очистка кэша приложения

Очистка кэша может пригодиться для устранения ошибок в работе Amnezia, а так
же для корректной настройки приложения.

На главном экране внизу кликните на название вашего сервера (на скриншоте он
обозначен как "Server2").

![](../../assets/images/cc_ru_1-9cdc8beac4ed1c83cb93d75caba2440e.png)

Выберите сервер, на котором вы хотите очистить кэш.

![](../../assets/images/cc_ru_2-fa54aef23d51bd6c970b866df7152a7c.png)

Во вкладке “Данные” выберите “Очистить кэш Amnezia”.

![](../../assets/images/cc_ru_3-545b7112cddf30f7b48b4370c4d9230c.png)
