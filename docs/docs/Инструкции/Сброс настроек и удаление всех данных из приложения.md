# Сброс настроек и удаление всех данных из приложения

На главном экране внизу кликните на значок настроек.

![](../../assets/images/r_ru_1-f6a91c222ab3f22eb035bbb1df5f6ed7.png)

Далее перейдите в раздел настроек, выберете “Приложение”.

![](../../assets/images/r_ru_2-b866d6cf007fe8dd52ac414e17ea430f.png)

На следующем экране выберете “Сбросить настройки и удалить все данные из
приложения”.

![](../../assets/images/r_ru_3-a04687c2bdbbd55984fcec4cca7c0e46.png)
