# Выбор языка

На главном экране внизу кликните на значок настроек.

![](../../assets/images/lc_ru_1-f6a91c222ab3f22eb035bbb1df5f6ed7.png)

Далее перейдите в раздел настроек “Приложение”.

![](../../assets/images/lc_ru_2-b866d6cf007fe8dd52ac414e17ea430f.png)

Выберите нужный язык.

![](../../assets/images/lc_ru_3-f864b762351339960cf7eeee0c51e026.png)
