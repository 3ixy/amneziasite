# Удаление сервера из приложения

Удаление сервера может понадобиться, если вы больше не хотите к нему
подключаться. При этом все установленные протоколы и сервисы Amnezia останутся
на сервере и будут доступны если вы снова добавите сервер в приложение.

Откройте приложение Amnezia

На главном экране внизу кликните на название вашего сервера.

На скриншоте он обозначен как "Server2".

![](../../assets/images/ds_ru_1-9cdc8beac4ed1c83cb93d75caba2440e.png)

Выберите сервер, который вы хотите удалить.

![](../../assets/images/ds_ru_2-347d98c00accf82b03d953a6ba812b6d.png)

Во вкладке “Данные” выберите “Удалить сервер из приложения”.

Чтобы удалить все протоколы и сервисы Amnezia с сервера нажмите “Удалить
сервисы Amnezia с сервера” перед тем как удалить сервер из приложения.

![](../../assets/images/ds_ru_3-9b1a9bd4d4b42a4c427852df111a2e31.png)
