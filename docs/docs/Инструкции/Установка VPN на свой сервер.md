# Установка VPN на свой сервер

## Установите приложение Amnezia[​](install-vpn-on-server.html#установите-
приложение-amnezia "Прямая ссылка на Установите приложение Amnezia")

На первом экране приложение спросит ваши данные для подключения, это данные
вашего VPS сервера. Если у вас нет VPS сервера, обратитесь к инструкции [Как
запустить свой VPN c помощью Amnezia](https://amnezia.org/starter-guide), там
собраны примеры инструкций для покупки хостинга и выбора нужных настроек.
После покупки VPS, хостинг-провайдеры отправят вам на почту данные IP, User
name или ID и пароль от сервера. Эти данные нам и нужны, они выглядят примерно
так:

  * **IP** : 192.565.ххx.xxx
  * **Username** : root (или другое)
  * **Password или SSH-key** : 2f9legf2...

Чтобы их внести откройте приложение AmneziaVPN, нажмите “Приступим”.

Если вы приобрели VPS для Amnezia у Inferno solution, вместо IP логина и
пароля вам могут отправить ключ в виде текста в таком формате
VPN:///ue34nf\feh..... В этом случае обратитесь к инструкции [Подключение
через ключ в виде текста](connect-via-text-key.html)

Нам нужно настроить VPN на собственном сервере, поэтому выбираем "Self-hosted
VPN"

![](../../assets/images/1sh_ru-108b9b0753595bbb3f1d6a169b15a5b8.png)

## Добавьте данные для подключения[​](install-vpn-on-server.html#добавьте-
данные-для-подключения "Прямая ссылка на Добавьте данные для подключения")

Внесите IP логин и пароль от VPS - сервера, нажмите “Продолжить”.

![](../../assets/images/2sh_ru-9a3f5beb7d163bbb3cd309b983a73852.png)

Если вы меняли port, укажите его через двоеточие после IP.

Если в ваших данных подключения нет логина, впишите root.

Вместо пароля, вы можете использовать SSH-ключ.

Вместо IP-адреса вы можете внести доменное имя.

## Выберите уровень контроля над интернетом в вашем регионе или перейдите во
вкладку с выбором протокола из списка[​](install-vpn-on-server.html#выберите-
уровень-контроля-над-интернетом-в-вашем-регионе-или-перейдите-во-вкладку-с-
выбором-протокола-из-списка "Прямая ссылка на Выберите уровень контроля над
интернетом в вашем регионе или перейдите во вкладку с выбором протокола из
списка")

Выбранный вами протокол можно будет в любой момент удалить или поменять, позже
можно будет установить любые другие поддерживаемые протоколы, этот шаг можно
пропустить.

При выборе **Высокого** уровня контроля будет установлен протокол AmneziaWG.
Это форк протокола WireGuard-GO с защитой от блокировок. Его отличительная
особенность - это измененные размеры Junk-пакетов и названия Magic-заголовков,
по которым, система DPI распознает протокол WireGuard. Эти параметры так же
можно изменить в настройках. Протокол работает в странах с самым высоким
уровнем интернет-цензуры.

При выборе **Низкого** уровня контроля будет установлен протокол WireGuard .
Подойдет для регионов, где не блокируются VPN-приложения и VPN-протоколы.
Повышает уровень приватности пользователя, и дополнительно шифрует трафик. Он
детектируется современными системами анализ трафика и заблокирован в некоторых
старанах с высоким уровнем интерент-цензуры.

Вы так же можете более подробно ознакомиться со всеми доступными протоколами в
[статье о протоколах на сайте](../protocols-info.html).

Если в вашем регионе заблокированы все неподконтрольные государству сайты и
практически все коммерческие и некоммерческие VPN, рекомендуем с первого
подключения выбрать **Высокий** уровень контроля, и не устанавливать другие
протоколы, так как в ином случае IP адрес вашего VPS может быть распознан и
заблокирован системами анализа и вы не сможете использовать его в будущем.

![](../../assets/images/3sh_ru-714edb0e00d16af37ac20077a678f153.png)

Дождитесь пока Amnezia настроит ваш VPN.

![](../../assets/images/4sh_ru-a757a1fccf877179884a91654b0c22d4.png)

Нажмите на кнопку **“Подключиться”**.

![](../../assets/images/5sh_ru-66ea0641c8bef51301b4704081587d02.png)
