# Перезагрузка сервера

Иногда, для отладки приложения или по другим причинам необходима перезагрузка
сервера. На главном экране кликните на название вашего сервера

![](../../assets/images/rs_ru_1-a433cfa9eff7c4432cbfaf3bd1f734b1.png)

Выберите сервер, который вы хотите перезагрузить,нажмите на шестеренку справа.

![](../../assets/images/rs_ru_2-e298bbeb86f1c379170b7f92804dded0.png)

Найдите вкладку “Управление” Далее выберите перезагрузить сервер

![](../../assets/images/rs_ru_3-e142d59c1d7971fadd094fd6f08877b0.png)
