# Предоставить полный доступ к VPN серверу

Вы можете предоставить права управления сервером через опцию "Поделиться
подключением", выбрав полный доступ.

Будьте осторожны: если пользователь удалит протокол или переустановит его, вы
не сможете подключиться к этому протоколу до его повторной установки. Поэтому
мы рекомендуем предоставлять полный доступ только своим устройствам.

  1. Откройте главный экран Amnezia

  2. На главном экране внизу кликните на значок поделиться.

![](../../assets/images/fc_ru_1-45e47c482c3d3bc7b0ac5435fb9e989d.png)

  3. Нажмите на “три точки” в правом верхнем углу экрана.

  4. Приложение предложит поделиться полным доступом к VPN, нажмите “Поделиться”. Далее убедитесь, что выбран нужный сервер и нажмите "Поделиться".

![](../../assets/images/fc_ru_2-4eacbefbe3889ebf0348cc1da67c1db0.png)

  5. Нажмите “Поделиться”, чтобы скачать файл конфигурации себе на устройство. Если вы делитесь с помощью QR-кода, отсканируйте его с помощью смартфона или планшета.

![](../../assets/images/fc_ru_3-40269f3c3cc7de17e468eed57236c308.png)
