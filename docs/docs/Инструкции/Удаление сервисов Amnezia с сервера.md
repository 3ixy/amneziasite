# Удаление сервисов Amnezia с сервера

При удалении сервисов Amnezia с сервера, будут удалены все установленные
протоколы, а так же все сервисы Amnezia, такие как сайт в сети Tor, файловое
хранилище и Amnezia DNS.

Откройте приложение Amnezia.

На главном экране внизу кликните на значок настроек.

![](../../assets/images/dsa_ru_1-b59589fa0652655e3295259e932d3215.png)

Далее перейдите в раздел “Серверы”.

![](../../assets/images/dsa_ru_2-b90d858e0104b39361c4c0fd16410077.png)

Выберите сервер.

![](../../assets/images/dsa_ru_3-cbc7a6d8c8dc836fd2f1f43b9894f83d.png)

Выберите вкладку “Данные”. Выберете “Удалить сервисы Amnezia с сервера”.

Все изменения на сервере, сделанные с помощью клиента Amnezia будут удалены.

![](../../assets/images/dsa_ru_4-ec783ddcdd52cdefe4aa2b5ec8b1e0f4.png)

страницаУдаление сервера из приложения](delete-server.html)
