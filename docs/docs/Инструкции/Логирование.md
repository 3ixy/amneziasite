# Логирование

Это функция сохранения логов. По умолчанию она отключена, но вы можете
включить ее, чтобы в случае неисправности работы приложения отправить файл с
логами в техническую поддержку.

На главном экране внизу кликните на значок настроек.

![](../../assets/images/l_ru_1-f6a91c222ab3f22eb035bbb1df5f6ed7.png)

Далее перейдите в раздел настроек “Приложение”.

![](../../assets/images/l_ru_2-b866d6cf007fe8dd52ac414e17ea430f.png)

Включите логирование.

![](../../assets/images/l_ru_3-a855abb2092138099818655d14230167.png)
