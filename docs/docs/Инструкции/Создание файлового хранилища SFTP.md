# Создание файлового хранилища SFTP

Помимо создание и использования VPN, с помощью Amnezia можно развернуть
файловое хранилище на вашем виртуальном сервере.

Откройте главный экран Amnezia. Внизу кликните на значок настроек.

![](../../assets/images/sftp_ru_1-f6a91c222ab3f22eb035bbb1df5f6ed7.png)

Далее перейдите в раздел “Серверы”.

![](../../assets/images/sftp_ru_2-63f59a1ad8a54359d9020abb9bb3b16f.png)

Выберете сервер, чтобы создать на нем файловое хранилище.

![](../../assets/images/sftp_ru_3-afbcc49107c1fbe593bf952f46f4c7d5.png)

Во вкладке "Сервисы" выберете “SFTP”.

![](../../assets/images/sftp_ru_4-bc3133c890dde8c1c080b76996064780.png)

Далее нажмите “Установить”.

![](../../assets/images/sftp_ru_5-c338bb935b4de71da3d5d3c6c89dc8c8.png)

После успешной установки вы увидите экран с настройками и данные для
подключения к файловому хранилищу.

Хост - это IP вашего сервера, порт обычно 22 или 222, логин и пароль.

С помощью этих данных можно получить доступ к файловому хранилищу через любой
SFTP-клиент.

![](../../assets/images/sftp_ru_6-e2f0c8529c9f013653659e1cfdd42f10.png)

В данном случае мы подключимся к хранилищу через
[FileZilla](https://filezilla-project.org/).

Вы можете пользоваться файловым хранилищем с помощью FileZilla и других
сторонних приложений, а можете [смонтировать директорию](sftp-mount-
folder.html) для хранилища на вашем компьютере.

Установите приложение. На первом же экране добавьте данные в строку как на
скриншоте.

Добавьте к вашему IP строку sftp://, получится примерно такой формат -
sftp://199.10.20.171 (с вашими цифрами), внесите эти данные в строку “Хост”,
остальные данные добавляем как есть. Нажмите “Быстрое соединение”.

![](../../assets/images/sftp_ru_7-1179bcf0b2fc5d2d4e49c989e627bbc6.png)

Далее в окне “Удаленный сайт” появится директория, а в статусе подключение
“Список каталогов извлечен” - это значит, что подключение прошло успешно и вы
можете пользоваться файловым хранилищем с помощью FileZilla.
