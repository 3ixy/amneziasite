# Альтернативные клиенты

В Amnezia есть возможность создать VPN, сохранить с ним конфигурацию и
использовать ее в других приложениях. Те есть VPN будет создан с помощью
клиента Amnezia а подключаться к VPN и использовать на постоянной основе вы
сможете через альтернативные клиенты, например через WIreGuard, OpenVPN
Connect, AmneziaWG и другие. Ниже представлены альтернативные клиенты для
некоторых протоколов.

### Клиенты для "AmneziaWG"

**AmneziaWG** \- Официальный клиент для протокола.

[Для Android в Google
Play](https://play.google.com/store/apps/details?id=org.amnezia.awg&hl=en_SG&gl=US)

[Для IOS в AppStore](https://apps.apple.com/pl/app/amneziawg/id6478942365)

Чтобы использовать VPN с AmneziaWG-протоколом в нативном приложении, после
установки протокола нажмите поделиться (на нижней панели), далее в
предложенном меню выберите формат подключения "Для AmneziaWG", сохраните
конфигурацию в виде файла на ваше устройство, после чего откройте конфигурацию
с помощью нативного приложения AmneziaWG.

Конфигурации с AmneziaFree не работают в нативном приложении - AmneziaWG,
используйте конфигурацию только с приложением AmneziaVPN.

### Клиенты для "OpenVPN over Cloak"

> Создать файл конфигурации, чтобы делиться подключением для клиента
> ShadowSocks и Shadowrocket можно только на 2 и 3 версии клиента, мы
> планируем добавить эту функцию в 4 версию Amnezia в ближайшем будущем.

Cloak - это плагин для обфускации трафика, он может использоваться с другими
протоколами. AmneziaVPN содержит в контейнере c OpenVPN over Cloak протокол
ShadowSocks для возможности использовать через клиент ShadowSocks.

Для ПК платформ:

  * **Windows** : "ck-client-windows", обычно есть два файла с пометками "386" (для 32-битной версии) и "amd64" (для 64 битной версии).
  * **Linux** : "ck-client-linux", имеется 10 файлов (по 32-битной и 64-битной версии) для архитектур x86, arm, mips, mipsle, mips_softfloat.
  * **MacOS** : "ck-client-darwin", есть сборка только для "amd64" (64-битных процессоров с архитектурой x86, не для новых процессоров Apple M1).

**Для Android** \- можно использовать в официальном Shadowsocks клиенте.

### Клиенты для "IPSec" (IKEv2)

**StrongSwan** \- кроссплатформенный клиент для IPSec IKEv2 для Windows,
Linux, FreeBSD, MacOS и Android.

Большинство современных устройств (кроме Android) имеет встроенную поддержку
IKEv2, и имеют возможность настройки подключения "из коробки". Для подключения
на них требуется знать IP-адрес сервера и иметь p12 сертификат.
