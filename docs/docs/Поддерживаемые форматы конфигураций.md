Приложение поддерживает несколько форматов конфигураций. Форматов конфигураций
- это формат файла, в котором Amnezia может сохранять или принимать настройки
вашего VPN.

**.vpn** \- нативный формат AmneziaVPN, файл с разрешением .vpn будет
сохранен, если вы нажмете поделиться VPN на нижней панели, далее выберите "Для
приложения AmneziaVPN"

**.ovpn** \- нативный формат для протокола OpenVPN, может использоваться в
пиложении OpenVPN, OpenVPN Connect и других приложениях, которые поддерживают
нативный формат протокола

**.conf** \- нативный формат для протокола WireGuard, может использоваться в
пиложении WireGuard, и других приложениях которые поддержтвают нативный формат
протокола.

**.json** \- поддерживается X-Ray Reality в формате .json, так же можно
импортироапть конфигурации с XRay VMESS и VLESS, однако нельзя создать VPN с
поддержкой протоклов VMESS и VLESS при помоши Amnezia.

**.json** \- поддерживаются .json ключи для ShadowSocks, но не поддерживаются
файлы с ShadowSocks в формате .json
